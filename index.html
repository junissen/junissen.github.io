About Me

After graduating from Vassar College with a B.A. in 2011, I spent time working as a clinical research coordinator, a cook, a rock climbing instructor, and an English teacher in Thailand. Upon my return from Southeast Asia, I completed a M.S. in Earth Sciences from the University of Minnesota in 2017. During my research, I created and implemented Python programs to streamline data reduction processes and improve data visualization, and found I loved coding! In 2017 I enrolled in the University of Minnesota's Coding Bootcamp, and aim to transition into a career in full stack development.

I have experience creating, developing, and deploying both front and back end applications. I am comfortable in HTML, CSS, JavaScript, jQuery, the MERN stack, Python and have worked with API inquiries and database communication.

When not working, I am an avid biker, yoga enthusiast, and dog lover. Most of this love is spent on my own fur baby, an Australian Cattle Dog. Additionally, I am a part of an Appalachian clogging group called Pomp and Circumstomp. Check us out on Facebook!

Data reduction for U/Th dating

GUI application that allows MC-ICP-MS users to reduce U/Th data and calculate ages from a central location, replacing former Excel methods and increasing productivity. Application created for use in the University of Minnesota Isotope Lab.

Travel On

Dynamic website that allows users to search a travel destination and access weather conditions and top points of interests, restaurants, and hotels.

Harry Potter Trivia Game

Dynamic Harry Potter trivia game in which users are prompted with questions and must answer within a specific time limit. If user answers correctly within the time limit, user score is increased. Otherwise, user score is decreased and correct answer is displayed.

Gif Generator

Dynamic website that allows users to search a 90s TV show to generate 10 gifs. Search will display still image and ratings, and upon clicking gif movement will be activated. Users can also add their own search terms or delete displayed gifs.

Train Schedule Application

Dynamic website that allows users to add train schedule details to the current train schedule. Train schedule calculates next arrival time and dynamically updates the time in minutes until the next arrival.

Liri Application

Node application that runs in command line that allows users to complete IMDB, Spotify, and Twitter searches using either prompts or txt files.

Hangman Application

Node application that runs in command line and allows users to play a game of hangman - 80s movies themed! User will be shown a blank word and prompted to guess letters. Game tracks guesses left, score, and letters guessed.

bAmazon Application

Node application that runs in command line works as a pared down Amazon storefront. Users can either view as a customer and buy products, or view as a manager and view, update, and/or add stock.

LIS Discharge Model

Python application that models MWP-1a discharge scenarios from southerly-routed Laurentide ice sheet melt.

Rayleigh fractionation Model

Python application that models Rayleigh fractionation from evaporation in the Gulf of Mexico to precipitation in Spring Valley Caverns, MN. Model recreates various scenarios for the Last Glacial Maximum and last interglacial.
